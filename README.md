### Para rodar a api

```
$ cd backend
$ npm i
$ npm run dev
```

Informe as variaveis de ambiente

```
PORT=[ porta que desejar rodar a api]
DB_URL="url mongodb"
```

### Para rodar o frontend

```
$ cd frontend
$ npm i
$ npm start
```
